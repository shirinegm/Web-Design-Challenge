Latitude vs. X

Summary

The data presented on this site is the result of an analysis using a city randomiser, selecting 500+ cities across the world and plotting several weather indicators against latitude. This is to try to predict which one of the weather indicators (Temperature, Cloudiness, Humidity or Winspeed) is affected by latitude. Use the cards on left to check out some cool in-depth analysis about each metric.
